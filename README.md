# twitter-chat-redirect
Redirects chat to messages on twitter.
# How to install
Install [TamperMonkey](https://www.tampermonkey.net) or [Violentmonkey](https://violentmonkey.github.io).
Then click [here](https://github.com/susstevedev/twitter-chat-redirect/raw/main/index.user.js). You should see an installation prompt. Click install.
